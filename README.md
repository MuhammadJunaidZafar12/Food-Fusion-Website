# Food Fusion Restaurant Website 🍔🍕

## Project Overview
Food Fusion is a simple and modern restaurant website developed as a **semester project during the 5th semester**.  
The purpose of this project was to practice **front-end web development** by creating a fast-food restaurant website using **HTML, CSS, and JavaScript**.

The website allows users to explore the restaurant menu, view special deals, and learn about the restaurant.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript

---

## Features
- Responsive and attractive homepage
- Fast-food menu section (Burgers, Pizza, Snacks, Drinks)
- Special offers section
- Food gallery
- Contact information page
- Simple and user-friendly interface

---


---

## Purpose of the Project
This project was created to:
- Understand website structure
- Practice HTML and CSS layout design
- Implement basic JavaScript functionality
- Gain practical experience in front-end development
